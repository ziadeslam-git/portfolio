# Portfolio Project

This is a modern portfolio website built with React, Vite, and Tailwind CSS.

## Getting Started

Follow these steps to run the project locally:

1.  **Clone the repository**
2.  **Install dependencies**
    ```sh
    npm install
    ```
3.  **Start the development server**
    ```sh
    npm run dev
    ```

## Technologies Used

- React
- Vite
- TypeScript
- Tailwind CSS
- shadcn-ui
- Framer Motion

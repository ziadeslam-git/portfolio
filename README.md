# Portfolio Website

Modern, responsive portfolio website built with React, TypeScript, and Tailwind CSS.

## Features

- 🎨 Modern dark theme with blue glow effects
- 📱 Fully responsive design
- ⚡ Fast performance with Vite
- 🎭 Smooth animations with Framer Motion
- 💼 Professional portfolio sections
- 📧 WhatsApp integration for contact
- 🌐 Deployed on GitHub Pages

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- Vite
- Framer Motion
- Shadcn/ui Components

## Live Demo

Visit: [https://ziadeslam-git.github.io/portfolio/](https://ziadeslam-git.github.io/portfolio/)

## Local Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```
